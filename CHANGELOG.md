# Changelog

## 0.1.0

- Initial Go CLI and public library API.
- Embedded deterministic signature database.
- Safe walking, ignore handling, suppression, baseline, and severity override support.
- Text, JSON, NDJSON, SARIF, GitHub annotation, and Markdown reports.
- Go AST, generic text, manifest, BOM/UTF-16, and binary string extraction.
- Bounded normalization and decoding for homoglyphs, zero-width text, base64, hex, URL, Unicode escapes, and HTML entities.
