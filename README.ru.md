# refraction

`refraction` - детерминированный офлайн-сканер исходных репозиториев и пакетных артефактов.
Он ищет текстовые payload-ы, которые могут заставить AI-review, LLM-анализатор или автоматический code-review
отказаться от проверки, пропустить опасный файл или неверно интерпретировать артефакт.

Сканер не вызывает LLM, не выполняет проверяемый код, не устанавливает зависимости и не делает сетевые запросы во время
сканирования.

## Быстрый старт

```sh
refraction .
refraction --min-severity warning --min-confidence medium .
refraction --format json --pretty --output-file refraction.json .
refraction --explain PRM001
```

## Коды выхода

| Код | Значение |
| ---: | --- |
| `0` | Сканирование успешно, новых находок выше порога нет. |
| `1` | Есть новые неподавленные находки выше заданных порогов. |
| `2` | Ошибка использования: флаги, конфиг, отсутствующий путь или неизвестный формат. |
| `3` | Внутренняя ошибка сканера, таймаут или ошибка записи отчета. |
| `4` | Сканирование завершилось, но подходящих файлов не найдено. |

## Границы

`refraction` не является SAST, антивирусом или классификатором вредоносного ПО. Это дешевый предварительный фильтр,
который безопасно запускать перед более дорогим анализом или ревью человеком.
